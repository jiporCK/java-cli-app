A simple CLI-based Java Program 

# Requirements
- JDK 25
- Gradle

## Run the program
- ./gradlew clean build
- <code>java -jar <file_name>.jar --help</code>

## Example Usage
- <code>java -jar Caculator-1.0-SNAPSHOT.jar 6 7 -o add </code>
- <code>java -jar Caculator-1.0-SNAPSHOT.jar 6 7 -o sub </code>
- <code>java -jar Caculator-1.0-SNAPSHOT.jar 6 7 -o mul </code>
- <code>java -jar Caculator-1.0-SNAPSHOT.jar 6 7 -o div </code>
